# Assistente de Troca SLM

🔗 Acesse o assistente:  
https://assistente-troca-slm.vercel.app/

O **Assistente de Troca SLM** é uma aplicação interna criada para apoiar o time operacional na análise de trocas de material didático Maple Bear.

A ferramenta automatiza a comparação entre o pedido principal e a nova compra, aplicando as regras de voucher, Pearson e juros, e gerando orientações claras sobre quando a troca pode seguir, quando é necessário complemento de valor e quando o processo deve seguir como cancelamento com reembolso.

---

## 📌 Visão Geral

O assistente foi desenvolvido para padronizar o processo de análise de trocas de material SLM, reduzindo erros manuais, retrabalho e tempo de resposta no atendimento.

Principais objetivos:

- Padronizar decisões operacionais sobre trocas de material
- Reduzir análises manuais
- Aumentar a agilidade no atendimento às escolas e responsáveis
- Garantir aplicação correta das regras de negócio
